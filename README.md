## goalscorer-arb-frotend
Goal scorer arb frontend will display goal scorer arbs. It's build with [Vue js](https://vuejs.org)

<p><a href="https://vuejs.org" target="_blank" rel="noopener noreferrer"><img width="70" src="https://vuejs.org/images/logo.png" alt="Vue logo"></a></p>

## Installation

* Clone the repo ` git clone https://github.com/AugustineTreezy/goalscorer-arb-frontend `
* `cd ` to project folder. 
* Run ` npm install ` 
* Run ` npm run serve ` 
* Done !!!
